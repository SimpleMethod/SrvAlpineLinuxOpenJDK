### SrvAlpineLinuxOpenJDK

A pseudo virtualization system based on Alpine Linux version 3.9.4. Access via noVNC

| PACKAGE | VERSION |
| ------ | ------ |
| Apache Maven | 3.6.0 |
| Gradle | 4.10.3 |
| openjdk | 1.8.0_212 |


To verify the operation of the system, navigate to the server address in your preferred browser.
```sh
127.0.0.1:8080
```
